Based on [Chisel](https://github.com/facebook/chisel).

## Usage

Add 

```
script fblldb.loadCommandsInDirectory('/path/to/lldb_commands/')
```

to your `.lldbinit` file after the `command script import <chisel>` line.

## Development

1. `command source ~/.lldbinit`
1. run command
1. rinse & repeat
